
cmd script to log onto raspberry and run py script to change times
putty_command.txt contains linux command that is excecuted on login

1. install putty/plink: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
2. create /Documents/AQI folder
3. put aqi_change_shelly_time.cmd and putty_command.txt in new AQI folder 
4. create desktop shortcut -> properties -> run minimized
5. set set_shelly_times.py to excecutable
